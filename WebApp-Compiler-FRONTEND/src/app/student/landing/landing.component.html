    General Instructions:
  1. Do not switch tabs or open other applications during the examination.
  2. Copy-paste is prohibited. You must enter your answers manually.
  3. Answer the Multiple Choice Questions (MCQs) first, and then proceed to programming questions.
  4. The total time remaining for the examination will be displayed in the navbar.
  5. Verify the details displayed in the navbar. If any discrepancies are found, inform the examiner immediately.
  6. Malpractice is recorded, and marks will be deducted accordingly.
  7. Read all instructions carefully before starting the examination.
    8. Programming Question:
  8. Save the program before submitting.
  9. You can run custom test cases using the run button.
  10. Enter custom input in the left box, and the output will be displayed here.
  11. Compilation errors will be shown in a separate window box.
  12. Select the programming language of the program.
  13. Do not press submit before saving the code.
  14. The Programming menu button in the left upper corner will show the list of programming questions.
  15. After submitting programming questions, click the submit button.
  16. Click the submit button before the time ends; otherwise, you will be automatically logged off, and your marks won't be recorded.
